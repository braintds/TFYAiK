Информация о программе
Программа написана в рамках первой лабораторной работы по дисциплине
"Теория формальных языков и компиляторов". Программа доработана в
рамках курсовой работы.
Техническое задание: Разработать приложение – текстовый редактор. В
дальнейшем он будет дополнен функциями языкового процессора.
Приложение имеет графический интерфейс пользователя. Язык реализации:
C#.
Текстовый редактор имеет следующие элементы:
1. Основное меню программы; o Пункт меню "Текст"; При вызове команд
этого меню должны открываться окна с соответствующей информацией по
курсовой работе
2. Панель инструментов; Панель инструментов содержит кнопки вызова
часто используемых пунктов меню: o Файл – Создать, Открыть, Сохранить; o
Правка – Отменить, Повторить, Вырезать, Копировать, Вставить; o Пуск;
Команда «Пуск» предназначена для запуска анализатора текста. o Вызов
справки. Приложение имеет справочную систему, запускаемую командой
«Вызов справки». Справка содержит описание всех реализованных функций
меню.
3. Окно/область ввода/редактирования текста; Область редактирования
представляет текстовый редактор. Команды меню “Файл" и "Правка”
работают с содержимым этой области.
4. Окно/область отображения результатов работы языкового процессора (в
этой области ввод текста запрещен).
В область отображения результатов выводятся сообщения и результаты
работы языкового процессора.
Интерфейс содержит дополнительные элементы и возможности:
1. Нумерация строк в окне редактирования текста.
2. Изменение размера текста в окне ввода и вывода
